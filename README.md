# Mobile Sales Dashboard 📊

## Project Overview
This project is an Excel-based Mobile Sales Dashboard created to analyze mobile sales performance using data visualization techniques.

## Objectives
- Analyze mobile sales data
- Track total sales and revenue
- Compare brand performance
- Monitor quantity sold
- Understand regional sales trends

## Tools Used
- Microsoft Excel
- Charts & Graphs
- Pivot Tables
- Dashboard Design

## Dashboard Features
✔ Total Sales Analysis  
✔ Revenue Tracking  
✔ Brand-wise Performance  
✔ Region-wise Sales Analysis  
✔ Data Visualization Dashboard
